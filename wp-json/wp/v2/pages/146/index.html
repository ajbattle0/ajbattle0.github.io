With boards predominately using buildings for terrain there can be an issue of boards feeling flat as the space inbetween buildings that models walk through street level is flat.<\/p>\n\n\n\n

This is where risers and platforms come in to be used as useful ways to elevate the battlefield. These pieces are kept flat ontop to allow for more terrain to be place on them and for models to traverse them easier. <\/p>\n\n\n\n
