    #1 Brief<\/h2>\n\n\n\n

    The first thing I did after speaking to Paul was write up the framework for a plan which I sent to Paul to make sure he felt comfortable working in this way:<\/p>\n\n\n\n

    Usually for these kinds of things I create a brief to explain my process. Here is what I was thinking as a frame work:\n1.\tA talk over the phone so that I can have the vision explained to me and have a natter.\n2.\tI will create a sketch for each panel of the comic.\n3.\tWe can have a feedback session where we talk about and refine the sketches towards something we are happy with.\n4.\tI\u2019ll draw each of the panels.\n5.\tWe have a feedback session for the drawn panels\n6.\tI\u2019ll colour each of the panels in a way similar to how I coloured these two projects project 1 project 2\n7.\tI\u2019ll finish the panel pages with any final details.\n8.\tThe panels will be sent to Katie as downloads from a google drive.\n9.\tKatie will put them together in the book and relay to me any changes that need to be made to have them look right in the publication.\n10.\tCelebrate with a couple drinks.\n<\/pre>\n\n\n\n
    I then went and spoke to Katie who was putting together the book to get some clear information on the timeline of the project and the details for the book:<\/p>\n\n\n\n
    A5 Pages.\nOur project will get 4 pages, 2 double spreads.\nThe deadline is for the new year.\nThe work will be printed using the risograph method.<\/pre>\n\n\n\n
