#7 Shading

Shading is a very difficult stage as there are many wrong ways to shade and few ways to do it right.

I setup my objects to be the scanned drawing layer first set to linear burn

My shading set to normal

My midtone set to normal.

All of this is set inside of a curves object.

This allowed me to get very consistent results. By working from the mid tone instead of the darkest tones I also saved a lot of time. As very often images would be around the midtone level anyway and so adding shading wasn’t convoluted.

Flowers:

After spending sometime trying out different styles I worked out that the style I was most happy with was using the chalk tool from dry media for the flowers

Skin

For skin often started by using shading brush (128 on the list) With this I would block out the larger shapes so the midtones, shadows and highlights. Next I would use shading detail brush (35 on the list) to add details particularly around creases. finally I would use the soft shading brush (35 on the list) to feather and blend some of the harsher transitions.

Brush list

Within fav brushes a place where I collect all the brushes I enjoy using.

Lighting

Finally some lighting effects were added using curve overlays. In these I created gradients using the transparency tool and then added these on top of the composition using the lighten blend mode.

#8 Typography

I began by looking at the typography of days of destruction. I quite liked the loose hand drawn feel of the text and so looked for something similar. I really liked the font Boldy Missy for this purpose.

I created small square boxes for the text to go in similarly to how they are done in sssscomics. By having some sides alittle imperfect it adds a much more readable feel as if the story is being told to you by a person rather than a robot.

#10 Updated brief

At this point the project brief was completed. However I had a meeting with the person who had commissioned the project and the decision was to convert this project to a standalone book in its own right.

The new plan was:

  1. Create a 12 page comic book by rearranging the panels as individual pages.
  2. Add a front cover and back cover.
  3. Add colour to the book.
  4. Have it prepared to print by a risographic printer.

With these new constraints I got to work.

#11 Rearranging the comic pages

The first thing I did was rearrange the pages so that each panel was its own page.

This was a very hard task as the assets were designed with a original intended layout and by altering the panels to pages assets had to be moved and rearranged with a few extra elements being drawn to fill some of the spaces made.

With these updates I also put some time into rearranging how I organised the project to make things neater.

#12 Adding colour

Next to add colour I created a flat shape for each of the different sections of colour. These were done by using the pen tool to follow the shape of the different sections. This was done to create a colour layer for each page.

This was a laborious process as each different section was drawn around.

#13 Converting the comic book for risographic printing

This was going to be printed using the risographic method where each of the colours would be printed separately by its own printing drum. This posed an interesting challenge as I would have to make it so that every colour printed would be able to use combinations of only 3 colours and black. So I would have to work out how the colours would combine and also a way to separate them out at the end to send to printing.

I achieved this by using CMYK. CMYK stands for Cyan Magneta, Yellow and Black. I coloured the book using the CMYK colours aswell as their combinations to give me access to more then just the three available colours.

One of the issues was the shadow layer. Each page had a detailed shadow I had drawn that I didn’t want to have to give its own separate layer to so I made it magenta so the magneta could also serve as the shading layer.

This was done by adding a half tone layer adjustment effect and then a colour shift to the shadow layer.

I then used channel mixer to remove all but one colour of each of the different CMYKs to create 4 versions of the comic with every layer separated out.

Here we can see how the layers look when added together and helps to explain how the risograph method works. It can be quite intimidating when working in this method as it can feel uncertain how the colours are going to look when they combine together however this test show that the experiment was successful.

#14 Creating the front and back pages

For the front and back pages I received some text and logos from the client, I had chose to use the deer skeleton for the front page as I feel like it was one of my my successful designs that explains the comic the best.

For the back page I went with the field of sunflowers the yetis were walking through but went with black and white to help it differentiate from the rest of the comic.
